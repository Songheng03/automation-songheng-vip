## ProductHunt — Upcoming Product

> Although ProductHunt uses a POST API method, the task requires a separate note here.

- **Website:** https://www.producthunt.com
- **Launch URL:** https://www.producthunt.com/launch
- **Type:** "Upcoming" product

### Instructions (Placeholder)
- Create a "Upcoming" product on ProductHunt via the web form at `/launch`.
- Prepare the following information:
  - **Name** — Product name
  - **Tagline** — Short, catchy tagline (up to ~60 characters)
  - **Description** — Detailed product description
  - **URL** — Product website URL
  - **Thumbnail Image** — High-quality logo/screenshot
  - **Topic Tags** — Select relevant topics/categories
- **API alternative:** ProductHunt also supports GraphQL API at `https://api.producthunt.com/v2/api/graphql` (requires Developer Bearer token).
- **Docs:** https://docs.producthunt.com/api
- **Note:** As an "Upcoming" product, you can gather followers before the official launch day. Schedule the launch for maximum visibility.
