# Show HN: Chaosong AI Gateway - 9 premium AI endpoints (free tier, self-hosted)

**Chaosong AI Gateway** is a lightweight, self-hosted API gateway that unifies access to 9 premium AI model providers through a single, simple interface.

## Features

- **9 AI endpoints** — OpenAI, Anthropic, Google Gemini, Mistral, Cohere, Groq, Perplexity, DeepSeek, and more
- **Free tier included** — Get started with a generous free tier, no credit card required
- **Self-hosted** — Full control over your data and costs
- **Single API key** — One key to rule them all
- **Load balancing & fallback** — Automatic failover between providers
- **Usage analytics** — Track spend and token usage across all providers

## Why build this?

Managing multiple API keys, SDKs, and billing accounts across different AI providers is a nightmare. Chaosong simplifies everything into one gateway with a unified API format.

## Try it

```
git clone https://github.com/chaosong/ai-gateway
cd ai-gateway && docker compose up
```

Read the docs at [docs.chaosong.ai](https://docs.chaosong.ai)
