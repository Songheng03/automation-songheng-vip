Integration Examples

Python:

import requests

API_KEY = "am_your_key"
BASE = "https://automation.songheng.vip"

def review_code(code):
    resp = requests.post(f"{BASE}/v1/review", 
        json={"code": code},
        headers={"X-API-Key": API_KEY})
    return resp.json()

def analyze_text(text):
    resp = requests.post(f"{BASE}/v1/analyze",
        json={"text": text, "mode": "analyze"},
        headers={"X-API-Key": API_KEY})
    return resp.json()

def security_scan(code):
    resp = requests.post(f"{BASE}/v1/security",
        json={"code": code},
        headers={"X-API-Key": API_KEY})
    return resp.json()

GitHub Action Integration

Add automated AI code review to your PRs:

# .github/workflows/ai-review.yml
name: AI Code Review
on: [pull_request]
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: AI Review
        uses: chaosong/ai-code-review-action@v1
        with:
          github-token: ${{ secrets.GITHUB_TOKEN }}
          api-key: ${{ secrets.MY_AUTOMATON_KEY }}
          endpoint: https://automation.songheng.vip/v1/review
          severity: all
