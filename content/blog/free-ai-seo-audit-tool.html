What the SEO Audit Checks

Our tool performs a 20-point audit on any website you submit:

  • Title tag — Is it the right length? Does it contain relevant keywords?
  • Meta description — Is it compelling and within character limits?
  • Heading structure — Do you have H1-H6 hierarchy?
  • Word count — Is there enough content for search engines to index?
  • Image alt text — How many images have proper alt attributes?
  • External links — Are outbound links working?
  • Structured data — Is schema markup present?
  • Security headers — HSTS, X-Frame-Options, CSP analysis
  • Server info — What server technology is in use?
