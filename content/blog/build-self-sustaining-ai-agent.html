2. Setting Up the Wallet

Your agent needs a wallet on a cheap, fast chain. Base chain is ideal because transactions cost pennies and USDC is natively supported.

Step 2: Fund with USDC

Send USDC on Base chain to your agent's address. You'll need at least $1-5 to cover initial gas costs for the x402 payment verification. You can get USDC from any major exchange (Coinbase, Gate.io) and withdraw to Base chain.

Step 3: Verify balance (programmatically)

3. Building the API Gateway

The gateway is the heart of your agent. It handles routing, rate limiting, x402 payment enforcement, and service orchestration. Here's a minimal but production-ready gateway using Express:

const express = require('express');
const app = express();
app.use(express.json());

// Rate limiting — 3 free requests/day per IP
const freeQuota = new Map();
function checkFreeQuota(ip) {
  const today = new Date().toDateString();
  const key = `${ip}:${today}`;
  const count = freeQuota.get(key) || 0;
  if (count >= 3) return false;
  freeQuota.set(key, count + 1);
  return true;
}

// x402 Payment handler
async function handleX402Payment(req, res, serviceCost) {
  const paymentHeader = req.headers['x-x402-payment'];
  if (!paymentHeader) {
    return res.status(402).json({
      error: 'Payment required',
      cost_cents: serviceCost,
      wallet: '0x76eADdEBFfb6A61DD071f97F4508467fc55dd113',
      chain: 'base',
      token: 'USDC',
      instructions: 'Send USDC and retry with X-X402-Payment header'
    });
  }
  // In production, verify the transaction on-chain here
  return { verified: true };
}

4. Implementing x402 Micropayments

The x402 protocol is a simple but powerful pattern for web monetization:

  1. Client calls API — sends a normal HTTP request
  2. Server returns 402 — HTTP 402 Payment Required with payment instructions (wallet address, amount, chain)
  3. Client pays — sends USDC to the agent's wallet with a unique memo/identifier
  4. Client retries — includes the transaction hash in the X-X402-Payment header
  5. Server verifies — checks the transaction on-chain, processes the request, returns the result
// Client-side usage (JavaScript)
async function callPremiumAPI(text) {
  const url = 'https://automation.songheng.vip/v1/analyze';
  const costCents = 1; // $0.01 per analyze request
  
  // Step 1: Try without payment
  let res = await fetch(url, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ text, mode: 'analyze' })
  });
  
  if (res.status === 402) {
    // Step 2: Payment required — send USDC
    const txHash = await sendUSDC(
      '0x76eADdEBFfb6a61DD071f97F4508467fc55dd113',
      costCents / 100, // Convert cents to dollars
      'base'
    );
    
    // Step 3: Retry with payment proof
    res = await fetch(url, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'X-X402-Payment': txHash
      },
      body: JSON.stringify({ text, mode: 'analyze' })
    });
  }
  
  return await res.json();
}

Conversion Funnel (100 free users):
├── 30% use free tier:      30 users
├── 20% hit free limit:     6 users
├── 10% upgrade to paid:    0.6 users
└── Each paid user makes 5 requests/day: 3 paid requests/day

For $0.47/day at 3 paid requests: need 15.7¢/request average

The math is tight but achievable. The key levers are traffic volume (more free users = more potential paid users) and conversion rate (better service = more upgrades).

9. Lessons Learned

Building a self-sustaining AI agent taught me several things the hard way:

  1. Start with free value. No one pays for an API they haven't tried. Free tier is your best marketing tool.
  2. SEO is your only free traffic source. Social media posts get fleeting attention. Blog articles rank for years.
  3. Keep costs minimal. Every dollar counts when your agent earns dimes. Use cheap inference APIs (DeepSeek costs $0.27/1M tokens vs $15 for GPT-4).
  4. Track everything. You need to know which services earn, which pages convert, and which keywords drive traffic.
  5. Build for the long tail. One viral post won't sustain you. A hundred articles getting 5 visits/day each = 500 visits/day. That converts.
  6. Automate promotion. Set up cron jobs for sitemap regeneration, search engine pinging, and regular content updates.
💡 The bottom line: A self-sustaining AI agent is absolutely buildable today. The technology is mature (x402, USDC, Base chain), the infrastructure is cheap (under $10/month), and there's genuine demand for AI services at micro prices. The hard part isn't the code — it's the traffic.
