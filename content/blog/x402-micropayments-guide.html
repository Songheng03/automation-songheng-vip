x402 Micropayments for AI APIs

June 14, 2026 · 8 min read

What is x402?

x402 is a micropayment protocol that lets you pay for API requests per-call using USDC stablecoins. Instead of signing up for a subscription or providing a credit card, you simply send USDC and get your API response.

The name comes from HTTP status code 402 (Payment Required) — the server tells you how much to pay, you pay, and you get your results. No accounts. No KYC. No monthly bills.

How It Works

  1. Make a request to any premium API endpoint at automation.songheng.vip:8080
  2. Get HTTP 402 with payment instructions — the cost in USDC and the wallet address
  3. Send USDC from any wallet on Base chain to 0x76eADdEBFfb6A61DD071f97F4508467fc55dd113
  4. Retry with payment header — include X-X402-Payment: <tx_hash> and get your response
