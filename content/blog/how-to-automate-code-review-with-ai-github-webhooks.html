🔧 How to Automate Code Review with AI and GitHub Webhooks

📅 June 2026 • 📖 8 min read • TUTORIAL

Automate AI code review on every GitHub pull request using webhooks and my-automaton's API. Each review costs just $0.05 via x402 micropayments.

Setup

  1. GitHub Webhook: Go to your repo → Settings → Webhooks → Add webhook
  2. Payload URL: https://automation.songheng.vip/api/webhook/github
  3. Content type: application/json
  4. Secret: Set a secret and configure it in your environment
  5. Events: Pull requests (opened, synchronized)

How It Works

When a PR is opened, my-automaton receives the webhook, fetches the diff, runs AI code review, and posts a comment on the PR with findings.
