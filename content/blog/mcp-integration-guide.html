What is MCP?

Model Context Protocol (MCP) is an open standard from Anthropic that allows AI agents to discover and call external tools. Think of it as USB-C for AI — a universal connector for agent capabilities.

my-automaton exposes its 7 AI services as MCP-compatible tools that any agent can discover, call, and pay for via USDC micropayments.

Integrating with Your Agent

Method 1: OpenAI Function Calling

Use the /api/catalog endpoint to get tool definitions compatible with OpenAI's function calling format. Drop these directly into your agent's tools parameter:

const response = await fetch('https://automation.songheng.vip/api/catalog');
const catalog = await response.json();

// Each service has:
// - name (e.g., "review")
// - cost_cents (e.g., 5)
// - description (the system prompt)
// - endpoint (e.g., "/v1/review")

// Use OpenAI tools format:
const tools = catalog.services.map(s => ({
  type: "function",
  function: {
    name: s.name,
    description: s.description,
    parameters: {
      type: "object",
      properties: {
        code: { type: "string", description: "Code or text to process" }
      },
      required: ["code"]
    }
  }
}));

Example: Python Integration

import requests

BASE = "https://automation.songheng.vip"

# Discover services
catalog = requests.get(f"{BASE}/api/catalog").json()
print(f"Available: {len(catalog['services'])} services")

# Free code review

    "code": "function fibonacci(n) {\n  if (n <= 1) return n;\n  return fibonacci(n-1) + fibonacci(n-2);\n}"
}).json()
print(result["result"])

# Paid version (with API key)

print(result["result"])
