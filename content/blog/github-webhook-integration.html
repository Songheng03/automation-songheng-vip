Setting up automated AI code review on every pull request saves your team hours of manual review. This guide walks through integrating my-automaton's AI code analysis with GitHub webhooks.

Manual code review is essential but time-consuming. AI-powered review:

  • Catches bugs and security vulnerabilities before merge
  • Enforces coding standards automatically
  • Provides consistent feedback across all PRs
  • Scales to any number of repositories
  • Costs pennies per review (5¢ for full security scan)

Prerequisites

Before starting, you'll need:

  • A GitHub repository (public or private)
  • Admin access to configure webhooks
  • A wallet with USDC on Base chain for premium features
  • Free tier available: 3 reviews/day without payment

Step 1: Create a GitHub Webhook

In your GitHub repository:

  1. Go to Settings → Webhooks → Add webhook
  2. Payload URL: https://automation.songheng.vip/webhooks/github
  3. Content type: application/json
  4. Secret: (optional but recommended — generates for API security)
  5. Select events: Pull requests, Pushes, Issues
  6. Click Add webhook
