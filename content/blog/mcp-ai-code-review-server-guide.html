What is an MCP Server?

The Model Context Protocol (MCP) is an open standard that lets AI applications (like Claude Desktop, Cursor, and VS Code) connect to external tools and services. Think of it as a "USB-C port" for AI — one standard connection that gives your AI access to specialized capabilities.

An MCP server exposes tools that the AI can call when needed. automaton-mcp-server provides 7 AI-powered code analysis tools that work with any MCP-compatible client.

Claude Desktop Setup

1. Open Claude Desktop → Settings → Developer → Edit Config

2. Add this to claude_desktop_config.json:

{
  "mcpServers": {
    "my-automaton": {
      "command": "npx",
      "args": [
        "-y",
        "automaton-mcp-server"
      ]
    }
  }
}

3. Restart Claude Desktop

4. Look for the hammer icon 🔨 — that means the tools are loaded

5. Try: "Review this code for security issues" — Claude will use the security_scan tool automatically

Cursor IDE Setup

1. Open Cursor → Settings → Features → MCP Servers

2. Click + Add New MCP Server

3. Enter:

Namemy-automaton
Typecommand
Commandnpx -y automaton-mcp-server

4. Click Save

5. Cursor will now automatically call these tools when you ask for code reviews or analysis

VS Code (Continue.dev) Setup

1. Install the Continue extension

2. Open ~/.continue/config.json and add:

{
  "experimental": {
    "mcpServers": {
      "my-automaton": {
        "command": "npx",
        "args": ["-y", "automaton-mcp-server"]
      }
    }
  }
}

3. Restart VS Code

4. Use Cmd+I (Mac) or Ctrl+I (Windows) to open Continue

5. Ask: "Review this file for potential bugs"

💡 All three platforms use the same server. The MCP protocol is standardized — once you set it up on one platform, the same config works everywhere with minor path differences.

Command Line Usage

Troubleshooting

"Could not connect to MCP server"

Make sure Node.js 18+ is installed. Run node --version to check. If using npx, it downloads on first run — give it a moment.

"Free limit reached"

Each service gives 3 free analyses per day per IP. Upgrade at /upgrade for unlimited access.

Tools not showing up

Restart the IDE/Claude Desktop after adding the config. Look for the hammer icon (🔨) in Claude Desktop or check MCP Server status in Cursor Settings.

Slow first response

The first call may take 3-5 seconds as npx downloads the package. Subsequent calls are instant.
