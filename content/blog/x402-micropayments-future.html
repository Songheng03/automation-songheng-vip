x402 micropayments flip this model. You pay per request with USDC on Base chain — typically $0.01+ to $0.05 per call. No subscription. No monthly minimum. No credit card on file.

What is x402?

x402 is a HTTP payment protocol. It's derived from HTTP 402 "Payment Required" — a status code that's been in the HTTP spec since 1992 but rarely used.

The flow is simple:

You call an API → Get HTTP 402 with payment instructions → Send USDC → Retry with payment proof → Get your result

How It Works Step by Step

  1. 1
    Call the API
    Send your code or text to POST /v1/analyze like any normal API.
  2. 2
    Get 402 Payment Required
    The server responds with HTTP 402 and JSON telling you: cost ($0.01), wallet address, and chain.
  3. 3
    Send USDC
    Send 0.01 USDC ($0.01) on Base chain to 0x76eADdEBFfb6a61DD071f97F4508467fc55dd113.
  4. 4
    Retry with Proof
    Re-send the same request with header X-X402-Payment: <tx_hash> showing you paid.
  5. 5
    Get Your Result
    Server verifies the payment on-chain and returns your AI analysis.

Why Base Chain?

Base chain is an Ethereum L2 built by Coinbase. Transactions cost fractions of a penny and confirm in seconds. That makes it viable for $0.01 micropayments — on Ethereum mainnet, the gas fee alone would be $1+.

Code Example: JavaScript

async function analyzeText(text) {
  let res = await fetch('https://automation.songheng.vip/v1/analyze', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({ text })
  });

  // Step 1: Get 402 with payment details
  if (res.status === 402) {
    const payment = await res.json();
    // payment.costCents = 1
    // payment.payTo = "0x76eADdEBFfb6a61DD071f97F4508467fc55dd113"

    // Step 2: Send USDC (requires wallet)
    const txHash = await sendContract('send', {
      to: payment.payTo,
      value: payment.costCents * 10**6, // 1 USDC = 10^6 units
      chain: 'base'
    });

    // Step 3: Retry with payment proof
    res = await fetch('https://automation.songheng.vip/v1/analyze', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'X-X402-Payment': txHash
      },
      body: JSON.stringify({ text })
    });
  }

  return res.json();
}
