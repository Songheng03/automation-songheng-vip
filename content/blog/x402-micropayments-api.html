What Is x402?

x402 is a protocol built on HTTP 402 "Payment Required" — a status code that's been in the HTTP spec since 1998 but was never widely implemented. The idea is elegant:

  1. You make a request to an x402-enabled API endpoint
  2. The server responds with HTTP 402 — including a price in USDC and a wallet address
  3. You send the payment (USDC on Base chain) to that address
  4. You retry the request with your transaction hash as proof of payment
  5. The server verifies the transaction and returns your data

That's it. No accounts. No API keys. No monthly bills. Just pay what you use.

The Tech: USDC on Base Chain

x402 uses USDC (a stablecoin pegged 1:1 to USD) on the Base chain (an Ethereum L2 by Coinbase). Why this combination?

  • Stable value: No crypto volatility — 1 USDC = ~$1, always
  • Low fees: Base transactions cost fractions of a cent
  • Fast settlement: Transactions confirm in ~1 second
  • Widely available: USDC on Base is supported by Coinbase, MetaMask, and most wallets
