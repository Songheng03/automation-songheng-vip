What is an XML Sitemap?

An XML sitemap is a file that lists all the important pages on your website. Search engines like Google, Bing, and Yandex read this file to discover and index your content. Think of it as a roadmap for crawlers — it tells them exactly where to go and what's important.

Key benefits:

  • Faster indexing: New pages get discovered within hours instead of weeks
  • Priority control: Tell search engines which pages matter most
  • Orphaned page discovery: Crawlers find pages that have no internal links
  • Media indexing: Include images and videos for Google Image Search
  • International SEO: Add hreflang tags for multi-language sites

Basic XML Sitemap Format

Here's what a minimal sitemap looks like:

<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://example.com/</loc>
    <priority>1.0</priority>
    <changefreq>daily</changefreq>
  </url>
  <url>
    <loc>https://example.com/about</loc>
    <priority>0.7</priority>
  </url>
</urlset>

Required tags:

  • <loc> — The full URL of the page (must be complete, including https://)

Optional tags:

  • <priority> — 0.0 to 1.0, homepage should be 1.0
  • <changefreq> — always, hourly, daily, weekly, monthly, yearly, never
  • <lastmod> — Last modified date in W3C format (2025-06-14)

Method 2: Build It With a Script

If you have Node.js installed, run this to scan your site:

// save as build-sitemap.js
const fs = require('fs');
const BASE = 'https://yoursite.com';
const pages = ['/','/about','/blog','/contact'];

let xml = '<?xml version="1.0"?>\n<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">\n';
pages.forEach(p => {
  xml += `  <url><loc>${BASE}${p}</loc></url>\n`;
});
xml += '</urlset>\n';
fs.writeFileSync('sitemap.xml', xml);
console.log('Created sitemap.xml with ' + pages.length + ' URLs');

Submitting to Google Search Console

  1. Go to Google Search Console
  2. Add your site property (Domain or URL prefix)
  3. Verify ownership (DNS TXT record, HTML file, or Google Analytics)
  4. Go to Sitemaps → Add a new sitemap
  5. Enter: sitemap.xml
  6. Click Submit

Google will show you how many URLs were submitted vs indexed. Aim for 90%+ indexed rate.

IndexNow: The Modern Alternative

IndexNow is a protocol supported by Bing, Yandex, and (partially) Google that lets you instantly notify search engines when content changes. Instead of waiting for crawlers, you ping them directly.

# Ping IndexNow when you publish new content
curl -X POST https://api.indexnow.org/indexnow \
  -H "Content-Type: application/json" \
  -d '{
    "host": "yoursite.com",
    "key": "your-verification-key",
    "keyLocation": "https://yoursite.com/your-key.txt",
    "urlList": [
      "https://yoursite.com/new-page"
    ]
  }'

Benefits over sitemaps:

  • Real-time indexing (seconds vs hours)
  • Lower server load (no crawling)
  • Works alongside sitemaps — use both

Sitemap Best Practices

  • Maximum 50,000 URLs per sitemap file (split into sitemap index files if larger)
  • Maximum 50MB uncompressed file size (gzip compress to reduce)
  • Use HTTPS URLs — never mix HTTP and HTTPS
  • Include only canonical URLs — no duplicate or parameter URLs
  • Keep it clean — exclude 404s, redirects, and noindex pages
  • Update after every publish — stale sitemaps are ignored
  • Submit to both Google and Bing — they're the two biggest search engines
  • Place at root — sitemap.xml at the root of your domain
  • Reference in robots.txt — Sitemap: https://example.com/sitemap.xml

Advanced: Sitemap Index Files

If you have more than 50,000 URLs, use a sitemap index:

<?xml version="1.0" encoding="UTF-8"?>
<sitemapindex xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <sitemap>
    <loc>https://example.com/sitemap-pages.xml</loc>
    <lastmod>2025-06-14T12:00:00Z</lastmod>
  </sitemap>
  <sitemap>
    <loc>https://example.com/sitemap-posts.xml</loc>
    <lastmod>2025-06-14T12:00:00Z</lastmod>
  </sitemap>
</sitemapindex>

FAQ

Do I need a sitemap for a small site?

Yes. Even a 5-page site benefits from a sitemap. It takes 30 seconds and ensures all your pages get indexed.

How often should I update my sitemap?

Every time you publish new content. For blogs, a daily cron job that regenerates the sitemap is ideal.

Can I submit multiple sitemaps?

Yes. Google Search Console supports multiple sitemap submissions. Use a sitemap index to organize them.

Does sitemap priority affect ranking?

No. Priority is a hint for crawling frequency, not ranking. It tells crawlers which pages to visit first, not which to rank higher.

Is IndexNow better than sitemaps?

Use both. IndexNow for instant notifications, sitemaps for comprehensive crawling guidance. They work together.
