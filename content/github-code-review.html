🚀 Install in 30 Seconds

1

Create the workflow file

Create .github/workflows/code-review.yml in your repo:

.github/workflows/code-review.yml 
name: Code Review

on:
  pull_request:
    types: [opened, synchronize]

jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: chaosong/my-automaton-review@v1
        with:
          github-token: ${{ secrets.GITHUB_TOKEN }}
          # Optional: add your API key for unlimited reviews
          # api-key: ${{ secrets.MY_AUTOMATON_KEY }}
          # mode: both  # 'review', 'security', or 'both' (default)
2

Push & open a PR

Commit and push the workflow file, then open a pull request. The AI will review your code automatically.

3

Read the review

Check your PR comments for AI-generated review feedback with bug detection, security alerts, and suggestions.

❓ FAQ

Is the free tier really free? No catch?
Yes. 3 reviews per day, per repository, forever. No credit card. No signup. No data collection. We make money when teams need more than 3 reviews/day and upgrade to a paid plan.
What's the difference between free and paid reviews?
Free reviews use our standard AI analysis. Paid reviews get priority processing, deeper analysis, and detailed CWE references. The AI model is the same — paid gets more depth and bandwidth.
Can I use this on private repositories?
Yes! The action runs in your CI environment. Your code is sent to our API for analysis. We don't store your code. For highly sensitive repos, contact us about self-hosted options.
How does it compare to CodeRabbit or CodeFactor?
We're simpler, cheaper, and pay-per-use. No monthly fees. No minimums. CodeRabbit starts at $12/month, CodeFactor at $15/month. We start at free and cost HK$38 for 500 reviews. If you do 10 reviews/month, we're HK$0.76 — vs $12+ elsewhere.
What if the review is wrong?
AI reviews are suggestions, not gospel. They're designed to catch obvious issues and complement (not replace) human review. Each comment includes a confidence indicator. Reviews get better as our AI model improves.
Can I customize what gets reviewed?
Yes! Set mode: review for code quality only, mode: security for security scanning only, or mode: both (default) for comprehensive review.
