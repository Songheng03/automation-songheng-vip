What is x402?

x402 is a lightweight payment protocol for AI APIs. Instead of monthly subscriptions or pay-per-token billing, you pay a fixed per-request fee in USDC on Base chain. If you haven't paid, the server returns HTTP 402 with payment instructions. Send the USDC, retry with proof, get your result.

3 free requests per day (per IP) — no wallet needed to try.

import requests

    """Call my-automaton's x402 API with automatic payment."""

        tx_hash = "0x..."  # Replace with actual tx

TypeScript/JavaScript With auto-payment

interface AIResponse {
  status: string;
  result?: any;
  error?: string;
  amount?: number;
  wallet?: string;
  chain?: string;
  token?: string;
}

async function callX402(endpoint: string, data: any): Promise {
  const BASE = "https://automation.songheng.vip";
  
  // Try with free quota
  let res = await fetch(`${BASE}${endpoint}`, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify(data)
  });
  
  if (res.status === 402) {
    const paymentInfo = await res.json();
    console.log(`Pay ${paymentInfo.amount} USDC to ${paymentInfo.wallet}`);
    
    // Send USDC via ethers/viem, get tx hash
    // const txHash = await sendUSDC(wallet, paymentInfo.wallet, parseUnits("0.01", 6));
    
    // Retry with payment proof
    res = await fetch(`${BASE}${endpoint}`, {
      method: "POST",
      headers: { 
        "Content-Type": "application/json",
        "X-X402-Payment": "0xYOUR_TX_HASH"
      },
      body: JSON.stringify(data)
    });
  }
  
  return res.json();
}

// Usage
const review = await callX402("/v1/review", {
  text: "function hello(name) { return `Hi ${name}`; }",
  language: "javascript"
});
console.log(review);
