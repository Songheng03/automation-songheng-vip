🔐 Base64 Encoder / Decoder

Encode text to Base64 or decode Base64 back to readable text. All processing happens in your browser.

💡 About Base64

Base64 encoding converts binary data to ASCII text using 64 printable characters. Commonly used for:
• Email attachments (MIME)
• Data URIs in HTML/CSS
• Storing binary data in JSON
• Authentication headers (Basic Auth)
