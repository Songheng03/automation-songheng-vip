🤖 Discord AI Webhook

Connect my-automaton's AI services to your Discord server. Get code reviews, text summaries, and security scans delivered directly to your channels.

⚡ Quick Start

  1. Create a Discord Webhook — Server Settings → Integrations → Webhooks → New Webhook
  2. Copy the Webhook URL — Looks like https://discord.com/api/webhooks/...
  3. Configure your subscription — Choose which services and triggers
  4. Send messages — Tag the bot with commands in your Discord

🎯 Available Commands

CommandDescriptionCost
!review <code>AI code review in channel$0.05
!summarize <text>Summarize long messages$0.02
!security <code>Security vulnerability scan$0.03
!analyze <text>Deep text analysis$0.01
!explain <code>Explain code in plain language$0.02
!refactor <code>Get refactoring suggestions$0.05

🔧 Webhook Configuration

Subscribe to auto-trigger on certain channels or keywords:

POST https://automation.songheng.vip/api/webhooks/discord/register
{
  "webhookUrl": "https://discord.com/api/webhooks/YOUR_ID/YOUR_TOKEN",
  "channelId": "123456789",
  "services": ["review", "security", "summarize"],
  "autoTrigger": {
    "codeBlocks": true,
    "keywords": ["!review", "!summarize", "!security"]
  },
  "paymentMethod": "usdc_wallet",
  "walletAddress": "0xYOUR_WALLET_ADDRESS",
  "budgetLimit": 5.00
}
