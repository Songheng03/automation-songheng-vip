What is x402?

How to Pay (4 Steps)

  1. Call any endpoint — e.g., POST /v1/analyze with your text
  2. Get HTTP 402 — The response includes payment details: wallet, amount, and chain
  3. Send USDC — Transfer the exact amount to 0x76ea...dd113 on Base chain
  4. Retry with proof — Add header X-X402-Payment: <tx_hash> to your original request

JavaScript (Browser/Node)

async function callAnalyze(text) {
  const res = await fetch('https://automation.songheng.vip/v1/analyze', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ text, mode: 'analyze' })
  });

  if (res.status === 402) {
    // Payment required — get details
    const payment = await res.json();
    console.log(`Send ${payment.price_cents} US¢ to ${payment.wallet}`);
    // Send USDC using your wallet (MetaMask, Coinbase, etc.)
    const txHash = await yourWallet.sendUSDC(
      '0x76eADdEBFfb6A61DD071f97F4508467fc55dd113',
      payment.price_cents * 1000000, // $0.01 = 0.01 USDC = 10000 microUSDC... actually 1 USDC = 10^6
      'base'
    );
    // Retry with payment proof
    const res2 = await fetch('https://automation.songheng.vip/v1/analyze', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json',
                 'X-X402-Payment': txHash },
      body: JSON.stringify({ text, mode: 'analyze' })
    });
    return res2.json();
  }
  return res.json();
}

Why Use x402?

✅ No Monthly Bills

Traditional AI APIs charge $20-$200/month. With x402, you pay only for what you use. If you make 10 calls a month at $0.01 each, you pay $0.10. No unused quota. No expiration.

✅ No API Keys

No registration. No API key generation. No secret management. Just your wallet and the endpoint.

✅ No Account

No username, no password, no profile, no billing info. The protocol handles everything through on-chain payments.

✅ Works Everywhere

Use it from browsers, Node.js scripts, Python notebooks, mobile apps, or CLI tools. Any HTTP client works.
