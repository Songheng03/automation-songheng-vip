🚀 Install my-automaton

AI code review, security scanning, text analysis — wherever you code

📦 Option 1: MCP Server (for Claude, Cursor, VS Code)

Connect AI-powered tools directly to your AI assistant or editor.

Manual install (no npm needed)

1
Download the server 
curl -O https://automation.songheng.vip/mcp-server.mjs
2
Add to Claude Desktop config

Edit ~/Library/Application Support/Claude/claude_desktop_config.json:

{
  "mcpServers": {
    "my-automaton": {
      "command": "node",
      "args": ["/path/to/mcp-server.mjs"]
    }
  }
}
3
Restart Claude

You'll see 7 new tools: code review, security scan, text analysis, summarization, code explanation, refactoring, and complexity analysis.

🤖 Option 3: GitHub Actions

Auto-review every PR with AI. 3 free reviews/day. FREE

# .github/workflows/ai-code-review.yml
name: AI Code Review
on: [pull_request]
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: AI Review
        run: |
          curl -s -X POST https://automation.songheng.vip/api/free/review \
            -H "Content-Type: application/json" \
            -d '{"code": "$(cat your-file.js)"}'
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
