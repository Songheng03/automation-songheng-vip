What is CIDR?

CIDR (Classless Inter-Domain Routing) notation represents IP address ranges compactly. 192.168.1.0/24 means the first 24 bits are the network prefix, leaving 8 bits for hosts.

Subnet Mask Explained

The subnet mask determines which part of the IP is the network and which is the host. /24 = 255.255.255.0. Each bit set to 1 in the mask is part of the network portion.

Usable vs Total

Usable hosts = Total − 2. The network address (all host bits 0) and broadcast address (all host bits 1) are reserved. A /24 has 256 total but 254 usable.
