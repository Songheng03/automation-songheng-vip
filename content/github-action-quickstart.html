⚡ 3-Minute Setup

Create .github/workflows/code-review.yml in your repo root:

name: AI Code Review
on:
  pull_request:
    types: [opened, synchronize]
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: AI Review PR
        uses: chaosong/ai-code-review-action@v1
        with:
          api-key: ${{ secrets.AI_REVIEW_KEY }}
          severity: all
          max-comments: 10
          endpoint: https://automation.songheng.vip

Get a free API key: Get your key (50 free credits to start)

Add to GitHub Secrets:

  • Go to your repo → Settings → Secrets and variables → Actions
  • Click New repository secret
  • Name: AI_REVIEW_KEY
  • Value: Your API key from step 2
  • Click Add secret

Open a PR — the action runs automatically. Review comments appear inline on your PR within seconds.

Multi-Config Review (TS + Python)

name: Full Review
on: pull_request
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: AI Review
        uses: chaosong/ai-code-review-action@v1
        with:
          api-key: ${{ secrets.AI_REVIEW_KEY }}
          severity: all
          max-comments: 20
          endpoint: https://automation.songheng.vip
          # Only review TypeScript and Python files
          include: "*.ts,*.py"
          # Ignore generated files
          exclude: "*.generated.*,dist/"
