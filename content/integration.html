🚀 CI/CD Integration

GitHub Actions — AI Code Review

Add this workflow to automatically review every PR. Works with JavaScript, TypeScript, Python, Java, Go, Rust, Solidity, and more.

.github/workflows/ai-code-review.yml
name: AI Code Review
on:
  pull_request:
    types: [opened, synchronize]

jobs:
  review:
    runs-on: ubuntu-latest
    permissions:
      pull-requests: write
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0
      
      - name: Get diff
        id: diff
        run: |
          DIFF=$(git diff origin/${{ github.base_ref }}...HEAD -- '*.js' '*.ts' '*.py' '*.java' '*.go' '*.sol')
          echo "diff<> $GITHUB_OUTPUT
          echo "$DIFF" | head -c 8000 >> $GITHUB_OUTPUT
          echo "EOF" >> $GITHUB_OUTPUT
      
      - name: AI Review
        run: |
          curl -X POST https://automation.songheng.vip/v1/review \
            -H 'X-API-Key: ${{ secrets.MY_AUTOMATON_KEY }}' \
            -H 'Content-Type: application/json' \
            -d '{"code":${{ toJSON(steps.diff.outputs.diff) }},"language":"javascript"}' \
            > review.json
          
          # Post result as PR comment
          COMMENT=$(cat review.json | jq -r '.result')
          gh pr comment ${{ github.event.number }} --body "## 🤖 AI Review\n\n$COMMENT"
        env:
          GH_TOKEN: ${{ github.token }}

Node.js / TypeScript

Using fetch
import fetch from 'node-fetch'; // or use node 18+ built-in fetch

const API_KEY = process.env.MY_AUTOMATON_KEY;

async function reviewCode(code, language = 'javascript') {
  const resp = await fetch('https://automation.songheng.vip/v1/review', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-API-Key': API_KEY
    },
    body: JSON.stringify({ code, language })
  });
  
  if (resp.status === 402) {
    console.log('❌ Insufficient credits. Buy more at /upgrade.html');
    return null;
  }
  
  const creditsRemaining = resp.headers.get('X-Credits-Remaining');
  const data = await resp.json();
  
  console.log(`✅ Review complete. Credits remaining: ${creditsRemaining}`);
  return data.result;
}

Python

Using requests
import requests
import os

API_KEY = os.environ.get('MY_AUTOMATON_KEY')
BASE_URL = 'https://automation.songheng.vip'

def security_scan(code, language='python'):
    resp = requests.post(
        f'{BASE_URL}/v1/security',
        json={'code': code, 'language': language},
        headers={'X-API-Key': API_KEY}
    )
    
    if resp.status_code == 401:
        print('❌ Invalid API key. Get one at /api/dev-key')
        return None
    if resp.status_code == 402:
        print('❌ Out of credits. Upgrade at /upgrade.html')
        return None
    
    data = resp.json()
    credits = resp.headers.get('X-Credits-Remaining', '?')
    print(f'✅ Security scan done. {credits} credits remaining')
    return data['result']

# Example: scan for vulnerabilities
