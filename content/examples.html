PREMIUM Batch Processing (10 texts)

Process up to 10 texts in a single API call for bulk analysis.

curl -X POST https://automation.songheng.vip/v1/batch \
  -H "Content-Type: application/json" \
  -d '{
    "texts": ["First article text...", "Second article...", "Third..."],
    "mode": "summarize"
  }'
const texts = ['Article 1...', 'Article 2...', 'Article 3...'];
const resp = await fetch('https://automation.songheng.vip/v1/batch', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ texts, mode: 'summarize' })
});
const results = await resp.json();
results.forEach((r, i) => console.log(`#${i+1}:`, r.summary));

FREE 🔄 x402 Micropayment Flow

The x402 protocol lets you handle the HTTP 402 response and automatically pay with USDC on Base chain.

async function callWithX402(url, data) {
  let resp = await fetch(url, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(data)
  });
  
  if (resp.status === 402) {
    const paymentInfo = await resp.json();
    console.log(`Payment required: ${paymentInfo.amount} USDC`);
    console.log(`Send to: ${paymentInfo.payer}`);
    
    // Send USDC on Base chain using your wallet
    const txHash = await sendUSDC(
      paymentInfo.payer,        // 0x76eADdEBFfb6a61DD071f97F4508467fc55dd113
      paymentInfo.amount,       // "0.05"
      'base'                    // Base chain
    );
    
    // Retry with payment proof
    resp = await fetch(url, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'X-X402-Payment': txHash
      },
      body: JSON.stringify(data)
    });
  }
  
  return resp.json();
}

Wallet: 0x76eADdEBFfb6a61DD071f97F4508467fc55dd113 (Base chain · USDC)
