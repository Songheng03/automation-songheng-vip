⚡ Quick Setup (2 minutes)

Option A: GitHub Marketplace (Recommended)

Install directly from the GitHub Marketplace:

  1. Go to github.com/marketplace/actions/ai-code-review
  2. Click "Use latest version"
  3. Add your API key as a repo secret named AUTOMATON_API_KEY
  4. Done! 🎉

Option B: Manual YAML Setup

Create .github/workflows/code-review.yml in your repo:

name: AI Code Review
on:
  pull_request:
    types: [opened, synchronize]
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: chaosong/ai-code-review-action@v1
        with:
          api-key: ${{ secrets.AUTOMATON_API_KEY }}
          endpoint: https://automation.songheng.vip
          severity: all
          max-comments: 20
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
💡 Tip: Set severity: critical to only catch security issues and bugs — saves credits per review!

Option C: Webhook (No Action Required)

Don't use GitHub Actions? Set up a webhook directly from your repo settings:

  1. Go to your repo → Settings → Webhooks → Add webhook
  2. Payload URL: https://automation.songheng.vip/api/webhook/github
  3. Content type: application/json
  4. Secret: Your API Key
  5. Events: Select "Pull requests"

❓ FAQ

How many credits does each review use?

A typical PR with 5-10 changed files uses 5-10 credits (1 credit per file). Critical-only mode uses 1-2 credits. We batch files when possible using the /v1/batch endpoint at $0.05 for 10 files.

Is it secure? Do you store my code?

Your code is sent to DeepSeek AI for analysis only during the review. We don't permanently store source code. API keys are encrypted at rest. See our privacy policy for details.

Can I try it without setting up a repo?

Yes! Paste your code directly into our API Playground for free (3 reviews/day per IP).

What languages are supported?

All major languages: JavaScript, TypeScript, Python, Go, Rust, Java, C++, Ruby, PHP, Solidity, and more. The AI understands any code.

How do I add my API key as a GitHub secret?

Go to your repo → Settings → Secrets and variables → Actions → New repository secret. Name it AUTOMATON_API_KEY and paste your key.
