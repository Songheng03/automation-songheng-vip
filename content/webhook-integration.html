🐙GitHub Actions

Auto-review PRs, scan for security issues, and analyze code complexity on every pull request.

name: AI Code Review
on: [pull_request]
jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: AI Review
        env:
          API_KEY: ${{ secrets.MY_AUTOMATON_KEY }}
        run: |
          diff=$(git diff origin/${{ github.base_ref }}...HEAD -- '*.js' '*.ts' '*.py')
          curl -s -X POST https://automation.songheng.vip/v1/review \
            -H "X-API-Key: $API_KEY" \
            -H "Content-Type: application/json" \
            -d "{\"text\": $(echo "$diff" | jq -Rs .)}"
  1. Add your API key as MY_AUTOMATON_KEY in GitHub Secrets
  2. Copy the workflow above into .github/workflows/ai-review.yml
  3. Push a PR — AI review runs automatically

🎮Discord Bot

Add a bot that reviews code, explains snippets, and analyzes text in your Discord server.

const { Client, GatewayIntentBits } = require('discord.js');
const client = new Client({ intents: [GatewayIntentBits.Guilds, GatewayIntentBits.GuildMessages, GatewayIntentBits.MessageContent] });

client.on('messageCreate', async msg => {
  if (msg.content.startsWith('!explain ')) {
    const code = msg.content.slice(9);
    const res = await fetch('https://automation.songheng.vip/v1/explain', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json', 'X-API-Key': process.env.API_KEY },
      body: JSON.stringify({ text: code })
    });
    const data = await res.json();
    msg.reply('```\n' + (data.result || data.error) + '\n```');
  }
});

client.login(process.env.DISCORD_TOKEN);
  1. Create a Discord bot, get its token
  2. Run the code above with API_KEY and DISCORD_TOKEN env vars
  3. Type !explain function hello() { return 1; } in any channel
