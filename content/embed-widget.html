🔧 How It Works

1. The widget adds a floating button to the bottom-right corner of your page.

2. When clicked, it opens a modal where users paste their code.

3. The code is sent to our AI API at automation.songheng.vip/v1/review.

4. Results appear instantly in the modal — security issues, best practices, performance tips.
