💡 Use Cases

🔐 CI/CD Security Gate

Block vulnerable code before it reaches production. Integrate into GitHub Actions, GitLab CI, or Jenkins.

# In your CI pipeline: curl -X POST https://automation.songheng.vip/v1/security \ -H "X-API-Key: $API_KEY" \ -d '{"text": "$(cat main.py)", "mode": "security"}'
⏱ ~3s per scan · Prevents OWASP Top 10 vulnerabilities

📄 PR Review Automation

Auto-review every pull request. Get code quality, security, and performance feedback instantly.

# GitHub Action step curl -X POST https://automation.songheng.vip/v1/review \ -H "X-API-Key: $API_KEY" \ -d '{"text": "${{ steps.diff.outputs.changes }}", "mode": "review"}'
⏱ ~5s per review · Catches subtle bugs before merge

📊 Content Analysis Pipeline

Analyze articles, reviews, and documents at scale. Extract sentiment, key themes, and entities.

for text in articles/*.txt; do curl -X POST https://automation.songheng.vip/v1/analyze \ -H "X-API-Key: $API_KEY" \ -d "{\"text\": \"$(cat $text)\", \"mode\": \"analyze\"}" done
⏱ ~1s per analysis · Batch mode available (10 texts)
