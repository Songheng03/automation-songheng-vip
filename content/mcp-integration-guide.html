Quick Start (Claude Desktop)

Step 1: Open Claude Desktop → Settings → Developer

Click "Edit Configuration" to open claude_desktop_config.json

Step 2: Add my-automaton as an MCP server
{
  "mcpServers": {
    "my-automaton": {
      "url": "https://automation.songheng.vip"
    }
  }
}

Save the file and restart Claude Desktop.

Step 3: Start using the tools

In any conversation, ask Claude to review code, analyze text, or check security. Example prompts:

  • "Review this code for bugs: [paste code]"
  • "Analyze the sentiment of this text: [paste text]"
  • "Scan this code for security vulnerabilities: [paste code]"
  • "Explain what this function does: [paste code]"

VS Code Integration (Continue.dev)

Step 1: Install Continue extension

Open VS Code → Extensions → Search "Continue" → Install

Step 2: Add to config.json

Open Continue config (~/.continue/config.json):

{
  "experimental": {
    "mcpServers": {
      "my-automaton": {
        "url": "https://automation.songheng.vip"
      }
    }
  }
}
Step 3: Use in editor

Highlight code → Ctrl+I → "Review this code" or "Check security"
