📦 Installation

Claude Desktop

Add to your claude_desktop_config.json:

{
  "mcpServers": {
    "my-automaton": {
      "url": "https://automation.songheng.vip/mcp/jsonrpc"
    }
  }
}

Cursor IDE

Add MCP server in Cursor settings → MCP Servers:

Name: my-automaton
Type: URL
URL: https://automation.songheng.vip/mcp/jsonrpc

Windsurf

Add to your MCP configuration:

{
  "mcpServers": {
    "my-automaton": {
      "url": "https://automation.songheng.vip/mcp/jsonrpc"
    }
  }
}
