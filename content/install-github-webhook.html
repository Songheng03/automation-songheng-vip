❓ FAQ

🔒 Is my code safe?
Yes. We only receive the diff (changes), not your entire repository. Diffs are processed in memory and not stored. We never share or train on your code.
🌐 Which languages are supported?
All major languages: JavaScript, TypeScript, Python, Java, Go, Rust, C/C++, Ruby, PHP, Swift, Kotlin, Solidity, and 50+ more. The AI auto-detects language.
📏 Is there a size limit?
Yes. Diffs over 50KB are skipped to ensure quick response times. For large PRs, consider splitting into smaller changes.
🔑 Can I use my own API key?
Yes! For production use, set up a premium API key. Configure the webhook with a secret and the gateway will use your key. Get API key →
📈 How does it compare to GitHub Copilot / CodeRabbit?
Free tier: 3 reviews/day vs CodeRabbit's paid-only model. Premium: ~$0.01/review vs CodeRabbit's ~$0.10/review. We focus on security, bugs, and performance — not just style.
