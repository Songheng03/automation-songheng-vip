🔗 GitHub + AI Code Review Integration

Connect your GitHub repositories to get automatic AI code reviews on every pull request and push. Security scanning, quality scoring, and style enforcement — all without leaving your workflow.

⚡ Quick Start (30 seconds)

  1. Go to your GitHub repo → Settings → Webhooks → Add webhook
  2. Payload URL: https://automation.songheng.vip/webhooks/github
  3. Content type: application/json
  4. Events: Send me everything (or select: Pushes, Pull requests, Issues)
  5. Click Add webhook ✅
