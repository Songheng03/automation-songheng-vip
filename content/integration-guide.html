🐙 GitHub Actions

Automatically review every pull request. Simply add this workflow to your repo.

Quick Setup (30 seconds)

Create .github/workflows/ai-code-review.yml in your repository:

name: AI Code Review on: pull_request: types: [opened, synchronize] jobs: review: runs-on: ubuntu-latest steps: - uses: actions/checkout@v4 - name: Get PR diff id: diff run: | curl -sL "${{ github.event.pull_request.diff_url }}" > pr.diff echo "size=$(wc -c < pr.diff)" >> $GITHUB_OUTPUT - name: AI Code Review id: ai_review run: | RESPONSE=$(curl -s -X POST https://automation.songheng.vip/api/free/review \ -H "Content-Type: application/json" \ -d "{\"text\": $(cat pr.diff | jq -Rs .), \"language\": \"auto\"}") echo "review=$RESPONSE" >> $GITHUB_OUTPUT - name: Post Review Comment uses: actions/github-script@v7 with: script: | const review = '${{ steps.ai_review.outputs.review }}'; github.rest.issues.createComment({ issue_number: context.issue.number, owner: context.repo.owner, repo: context.repo.repo, body: "## 🤖 AI Code Review\n\n" + review + "\n\n---\n_Powered by [my-automaton](https://automation.songheng.vip)_" })

💻 CLI Tool (curl)

Review files directly from your terminal. Works with any language.

Review a File

curl -X POST https://automation.songheng.vip/api/free/review \ -H "Content-Type: application/json" \ -d '{"text": "function add(a,b){return a+b}", "language": "javascript"}'

Pipe a File

cat your_file.js | jq -Rs '{text: ., language: "javascript"}' | \ curl -X POST https://automation.songheng.vip/api/free/review \ -H "Content-Type: application/json" -d @-

💬 Slack Bot

Review code directly from Slack. Drop a code snippet in any channel and get instant AI feedback.

Setup

  1. Go to Slack API → Create New App
  2. Enable Slash Commands → Create command /review
  3. Set Request URL to https://automation.songheng.vip/api/slack/review
  4. Install the app to your workspace

Usage

/review function add(a,b){return a+b} — Reviews JavaScript code /review --lang python def add(a,b): return a+b — Specify language /review --help — Shows available options
