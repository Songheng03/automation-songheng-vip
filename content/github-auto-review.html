⚙️ How It Works

  1. You buy credits and get an API key
  2. You connect a repo and provide a GitHub token
  3. We install a webhook on your repo: POST /api/github-webhook
  4. Every time a PR is opened or updated, we automatically run: code review + security scan + complexity analysis
  5. Results are posted as a PR comment within 30 seconds
  6. Each review costs 5 credits ($0.05) from your balance
