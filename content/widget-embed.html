⚙️ Configuration Options

Customize the widget via data-* attributes on the script tag:

data-theme       = "dark" | "light"        (default: dark)
data-accent      = any hex color            (default: #6366f1)
data-btn-text    = custom button text       (default: 🚀 Analyze Code)
data-placeholder = custom placeholder       (default: Paste your code...)
data-site        = your site name           (for analytics)
