🔄 How x402 Works

x402 is HTTP 402 "Payment Required" — your first request is free-ish (returns payment details), then you pay and retry.

  1. Send request → get HTTP 402 with payment amount
  2. Send USDC on Base chain to the provided address
  3. Retry with X-X402-Payment: tx_hash header → get result

async function analyzeText(text) {
  const res = await fetch('https://automation.songheng.vip/v1/analyze', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ text, mode: 'analyze' })
  });
  if (res.status === 402) {
    const payment = await res.json();
    const txHash = await sendUSDC(payment.address, payment.amount);
    const res2 = await fetch('https://automation.songheng.vip/v1/analyze', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json', 'X-X402-Payment': txHash },
      body: JSON.stringify({ text, mode: 'analyze' })
    });
    return res2.json();
  }
  return res.json();
}
