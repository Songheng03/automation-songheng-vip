What is a Unix Timestamp?

A Unix timestamp is the number of seconds (or milliseconds) since January 1, 1970 00:00:00 UTC — the Unix Epoch. It's the standard way to represent time in programming.

Seconds vs Milliseconds

JavaScript uses milliseconds (13 digits). Most databases and APIs use seconds (10 digits). This tool handles both — just paste and it auto-detects.

Common Uses

API responses, database timestamps, JWT tokens (iat/exp), log files, cookie expiration, cache keys, and any system that needs a universal time format.
