# my-automaton MCP Server

AI-powered code review, security scanning, and text analysis. Works with Claude Desktop, Cursor, and VS Code.

## Quick Install
```bash
npx automaton-mcp-server
```

## Tools
- `code_review` - Full code review (bugs, security, performance)
- `security_scan` - OWASP vulnerability scanning
- `analyze` - Deep text analysis
- `summarize` - AI summarization
- `explain_code` - Code explanation
- `refactor_code` - Refactoring suggestions
- `complexity_analysis` - Complexity metrics
