❓ FAQ

What is MCP?

Model Context Protocol (MCP) is an open standard for AI apps to discover and use tools. Claude Desktop, Cursor, and VS Code all support it natively.

Is it really free?

Yes! 3 analyses per tool per day, completely free. Need more? API keys start at $5 for 500 credits.

What tools are available?

Code review, security scanning, text analysis, summarization, code explanation, refactoring suggestions, and complexity analysis.

How do I use this in Cursor?

Open Cursor Settings → Features → MCP Servers → "Add New MCP Server" → Paste the config from this page. Done.
