📋 How to Set Up

Get an API Key

Choose a plan and pay via x402 USDC. You'll receive an am_xxx API key immediately.

🔑 Get API Key

Add GitHub Actions Workflow

Create .github/workflows/ai-code-review.yml in your repo:

# .github/workflows/ai-code-review.yml name: AI Code Review
on:
  pull_request:
    types: [opened, synchronize]

jobs:
  review:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: AI Code Review
        run: |
          curl -s -X POST https://automation.songheng.vip/v1/review \
            -H "Content-Type: application/json" \
            -H "X-API-Key: ${{ secrets.MY_AUTOMATON_KEY }}" \
            -d '{"text": "'$(cat ${{ github.event.pull_request.diff_url }} | base64)'"}'

Add Secret to GitHub

Go to your repo → Settings → Secrets and variables → Actions

Add a new secret: MY_AUTOMATON_KEY with your am_xxx key.

Open a PR

That's it! Every PR will get an AI code review comment automatically.

Using free tier? 3 reviews/day included.
