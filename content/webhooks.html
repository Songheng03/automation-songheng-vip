🐙 GitHub Integration

Automatically review PRs, analyze code, and get security insights right in your GitHub workflow.

Setup

  1. Go to your repo → Settings → Webhooks → Add webhook
  2. Payload URL: https://automation.songheng.vip/api/webhooks/github
  3. Content type: application/json
  4. Secret: (optional) Set a secret for verification
  5. Events: Select Pull requests, Pushes, Issues
  6. Click Add webhook

Supported Events

pull_request
Auto-analyze new PRs and code changes
push
Track commits and branch updates
issues
Monitor issue creation and updates
💡 Pro
Use AI code review: POST /v1/review with PR diff

💬 Slack Integration

Use slash commands to analyze text, review code, and summarize content directly from Slack.

Setup

  1. Go to api.slack.com/apps → Create New App
  2. Enable "Slash Commands" and add commands below
  3. Request URL: https://automation.songheng.vip/api/webhooks/slack
  4. Install the app to your workspace

Slash Commands

/analyze <text>
AI-powered text analysis and insights
Free
/review <code>
Code review with suggestions
Free
/summarize <text>
Summarize long content
Free
/help
List available commands
Free

🎮 Discord Bot

Add AI analysis bots to your Discord server with slash commands.

Setup

  1. Go to Discord Developer Portal
  2. Create Application → Bot → Copy token
  3. Set Interactions Endpoint URL: https://automation.songheng.vip/api/webhooks/discord
  4. Invite bot to your server with applications.commands scope

Slash Commands

/analyze text:<text>
Deep text analysis with AI
/review code:<code>
Comprehensive code review
/help
List all available commands
💡 Pro Tips
Use code blocks for multi-line input
