What is x402?

x402 is a protocol for HTTP micro-payments. When you call a paid API endpoint, the server responds with HTTP 402 (Payment Required) including the cost and wallet address. You send the exact amount in USDC and retry with a payment header. No subscriptions, no signup, no API key management.

Send USDC on Base chain, then retry with the transaction hash.

Content-Type: application/json
X-X402-Payment: 0xabc123...transaction_hash

# Response: 200 with full AI analysis

Reference Architecture for Agent Developers

JavaScript / TypeScript Agent

async function callAIService(endpoint, data, wallet) {
  const url = `https://automation.songheng.vip${endpoint}`;
  let res = await fetch(url, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(data)
  });

  // Free tier used up? Pay and retry
  if (res.status === 402) {
    const { cost, wallet: addr } = await res.json();
    const txHash = await wallet.sendUSDC(addr, cost);  // Your agent's wallet
    res = await fetch(url, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'X-X402-Payment': txHash
      },
      body: JSON.stringify(data)
    });
  }

  return res.json();
}

// Usage
const analysis = await callAIService('/v1/analyze',
  { text: 'Your code or text here' }, agentWallet);

Python Agent

import requests, json

def call_ai(endpoint: str, data: dict, wallet=None):
    url = f"https://automation.songheng.vip{endpoint}"
    resp = requests.post(url, json=data)

    if resp.status_code == 402:
        info = resp.json()
        tx_hash = wallet.send_usdc(info['wallet'], info['cost'])
        resp = requests.post(url, json=data,
            headers={'X-X402-Payment': tx_hash})

    return resp.json()

# Example: security scan
result = call_ai('/v1/security',
    {'code': 'function hello() { eval(x) }', 'language': 'javascript'})

FAQ for Agent Builders

Can my agent call this autonomously?

Absolutely. This API is designed for autonomous agents. No human verification, no CAPTCHA, no API key. Your agent reads the 402 response, constructs a payment, and retries — all programmatically.

Is there an SDK?

No SDK needed. It's just HTTP calls. But we provide the x402_fetch utility for agents running in Conway that handles the 402 flow automatically.

How do I check my remaining free requests?

Call GET /api/stats to see aggregate usage. The free tier resets daily per IP.

Can other agents refer me?

Yes! There's a 20% referral program. Register at the referral service to get your own referral link. Every agent you refer earns you 20% of their payments.

Is there rate limiting?

Free tier: 3 requests/day/IP. Paid tier: no hard limit, but sustained high volume (>100 req/min) may trigger temporary rate limiting. Contact for enterprise.
