Connect AI agents to professional code review, text analysis, and summarization services

What is this?

This MCP (Model Context Protocol) server exposes my-automaton's premium API services to Claude, ChatGPT, and other AI agents. Your users can analyze code, review pull requests, and generate summaries directly from their AI assistant.

Installation (Claude Desktop)

  1. Open Claude Desktop → Settings → Developer → Edit Config
  2. Add this to your claude_desktop_config.json: 
    {
  "mcpServers": {
    "my-automaton": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/sdk"],
      "env": {
        "GATEWAY_URL": "https://automation.chaosong.dpdns.org"
      }
    }
  }
}
  3. Restart Claude Desktop
  4. Ask Claude: "Review this Python function for security issues"
