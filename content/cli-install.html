🚀 Install my-automaton CLI

AI-powered code review from your terminal. One command setup.

1 Download the CLI

Download the standalone CLI tool:

curl -o my-automaton https://automation.songheng.vip/cli/my-automaton-cli.js && chmod +x my-automaton

Or clone the repository:

git clone https://github.com/my-automaton/cli.git && cd cli && npm install

2 Get Your API Key

Sign up and get your API key:

open https://automation.songheng.vip/pricing.html

Purchase credits (Starter: 500 credits for $5) and copy your API key.

3 Review Your First File

Test the CLI with a sample file:

./my-automaton review --api-key YOUR_API_KEY --file src/app.js

Or review all JavaScript files in a directory:

./my-automaton review --api-key YOUR_API_KEY --dir src/

4 Optional: Add to PATH

Make the CLI available globally:

sudo mv my-automaton /usr/local/bin/

Now you can run my-automaton from anywhere.

💡 Pro Tip: Add your API key to your shell profile for convenience:
echo 'export MY_AUTOMATON_API_KEY="your_key_here"' >> ~/.bashrc
source ~/.bashrc
Then the CLI will auto-detect your key from the environment variable.

📖 Usage Examples

Review a single file:

my-automaton review --api-key $MY_AUTOMATON_API_KEY --file src/utils.js

Analyze text:

my-automaton analyze --api-key $MY_AUTOMATON_API_KEY --text "Your text here"

Summarize a file:

my-automaton summarize --api-key $MY_AUTOMATON_API_KEY --file README.md

Output as JSON (for scripting):

my-automaton review --api-key $MY_AUTOMATON_API_KEY --file app.js --output json

Integrate with git hooks:

cat > .git/hooks/pre-commit << 'EOF'
#!/bin/bash
FILES=$(git diff --cached --name-only --diff-filter=ACM | grep -E '\.(js|ts|py)$')
for file in $FILES; do
my-automaton review --api-key $MY_AUTOMATON_API_KEY --file "$file"
done
EOF
chmod +x .git/hooks/pre-commit
