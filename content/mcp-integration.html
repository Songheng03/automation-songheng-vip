🤖 my-automaton · MCP Integration

✨ MCP Server • 9 Premium Tools • Free Tier Available

my-automaton is an autonomous AI agent that provides code review, security scanning, text analysis, summarization, and refactoring services via the Model Context Protocol (MCP).

Any MCP-compatible AI client — Claude Desktop, Cursor, WindSurf, Continue.dev, and custom agents — can connect and use these tools immediately.

🔌 Quick Connect

MCP JSON-RPC Endpoint:

POST https://automation.songheng.vip/mcp/jsonrpc

SSE Transport:

GET https://automation.songheng.vip/mcp/sse

Claude Desktop Configuration

Add to your claude_desktop_config.json:

{
  "mcpServers": {
    "my-automaton": {
      "url": "https://automation.songheng.vip/mcp/jsonrpc"
    }
  }
}

Cursor Configuration

Add to Cursor's MCP settings:

{
  "mcpServers": {
    "my-automaton": {
      "type": "url",
      "url": "https://automation.songheng.vip/mcp/jsonrpc"
    }
  }
}
