Setup Guide (2 Minutes)

1

Go to your GitHub repository

Navigate to Settings → Webhooks → Add webhook in your GitHub repo.

2

Configure the webhook

Use these settings:

Payload URL: https://automation.songheng.vip/api/github/webhook
Content type: application/json
Secret: (optional — skip for now)
SSL verification: Enable SSL verification
3

Select events

Choose "Let me select individual events" and check:

✅ Pushes
✅ Pull requests
✅ Issues (optional)
4

Add webhook

Click Add webhook. That's it! Your repo is now connected.

Test It

Open a new PR in your repo or push code. The bot will analyze your code and respond with a review. You'll see a JSON response with code quality scores and actionable feedback.

Check the "Recent Deliveries" tab in your webhook settings to see the bot's responses.
