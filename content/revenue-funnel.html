Frequently Asked Questions

Do I need a credit card to start? ↓

No! You get 3 free reviews per day per IP address. Just call the /free/ endpoints to try it out. When you're ready, choose a plan to get your own API key.

How do credits work? ↓

1 credit = $0.001 USD approximately. Each API endpoint costs 1-5 credits per call. Once you purchase credits, they never expire. Your balance is shown in your API key dashboard.

What languages do you support? ↓

All major languages: JavaScript/TypeScript, Python, Rust, Go, Solidity, Java, C++, Ruby, PHP, and more. The AI understands code semantics, not just syntax.

Can I use this in CI/CD? ↓

Absolutely! GitHub Actions, GitLab CI, CircleCI, Jenkins — any CI that can make HTTP requests. Store your API key as a secret and add the review step to any pipeline.

What if I run out of credits? ↓

Buy another plan! Credits never expire and roll over. The API returns HTTP 402 when your credits are exhausted, so your CI pipeline fails gracefully.
